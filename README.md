# Excavation Company Website

A simple React + Vite showcase website for an excavation company. It includes:

- Home hero section
- Services section
- Photo gallery
- About section
- Contact section
- Responsive mobile navigation

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Deploy

Push this folder to GitHub, then import the repo in Vercel.

Vercel settings:

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`

## Customize

Edit `src/main.jsx`:

- Company name
- Phone number
- Email
- Service area
- Services
- Gallery images

Edit `src/styles.css` to change colors and layout.
